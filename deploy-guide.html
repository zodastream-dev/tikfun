☁️推荐静态托管方案

🥇 Vercel（最推荐）免费全球CDN

支持自定义域名、HTTPS自动、全球加速，5分钟内完成部署。

🥈 Netlify免费

同样支持拖拽上传，自定义域名，HTTPS自动签发。

🥉 腾讯云 COS + CDN国内加速

适合国内用户访问，需要域名备案（yookeer.com 建议提前确认备案状态）。

⚡Vercel 部署步骤（推荐）

1

访问 Vercel 并登录

打开 vercel.com，使用 GitHub / Google 账号登录

2

创建新项目 → 选择"Browse"上传文件夹

点击 Add New → Project，选择 Upload，将 tikfun/ 文件夹拖入

3

部署配置（零配置即可）

Framework Preset 选 Other，Root Directory 保持默认，直接点击 Deploy

4

绑定自定义域名

部署成功后进入 Settings → Domains，添加 workbuddy.yookeer.com

5

配置 DNS（在域名注册商处操作）

Vercel 会给出 CNAME 目标值，在 yookeer.com 的 DNS 面板添加以下记录：

🌐DNS 配置

类型 主机记录 记录值 TTL
CNAME workbuddy cname.vercel-dns.com 300

⏱ DNS 生效时间：通常 5-30 分钟，最长 48 小时。Vercel 控制台会自动检测并签发 HTTPS 证书。

🇨🇳腾讯云 COS 部署步骤（备选）

1

创建 COS Bucket

进入 腾讯云控制台 → 对象存储 → 创建存储桶，地域选广州或上海

2

上传文件

上传 index.html 和 data/videos.js（注意保持目录结构）

3

开启静态网站托管

存储桶 → 基础配置 → 静态网站，开启并设置 索引文档为 index.html

4

绑定自定义域名 + CDN

域名管理 → 添加 workbuddy.yookeer.com，开启 CDN 加速，配置 HTTPS（免费 DV 证书）

⚠️ 备案提醒：腾讯云国内 CDN 需要域名完成 ICP 备案。请提前确认 yookeer.com 的备案状态；如未备案，建议使用 Vercel 方案。
